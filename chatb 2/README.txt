Chat B — artefatos
==================
divisões do tempo e do espaço · tik · tak

Como usar
---------
Abra index.html no navegador para navegar e buscar os artefatos.
Todos os arquivos funcionam localmente, sem servidor.

Arquivos
--------
index.html                — galeria de busca (ponto de entrada)
tik-tak-som.html          — tik·tak tela inteira com som
compasso-cima-baixo.html  — compasso com linha do tempo e rastro
seis-canais.html          — 6 canais crescentes simultâneos
faixas-numeradas-v1.html  — faixas numeradas h/v, 3 modos
faixas-numeradas-v2.html  — faixas numeradas com pulso no número
faixas-tik-tak-som.html   — faixas + bolas + som (versão completa)

Tags
----
ritmo   — objetos que batem no tempo
som     — usa Web Audio API (requer toque para ativar)
faixas  — pano de fundo com divisões de cor
visual  — sem som, foco visual
grade   — combinação de múltiplos elementos
