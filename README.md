# Jogo da Vida

Sistema de duas plataformas para saúde digital e lucidez.

## Estrutura

- **especificador/** - dimensão 1: inspiração, sessões internas, autoconhecimento
- **cores/** - dimensão 2: prótese digital, uso em movimento via celular
- **shared/** - recursos compartilhados: personas, grafos, imagens
- **dev/** - desenvolvimento: scripts Python, documentação
